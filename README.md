# WhatsApp DMs
DM WhatsApp users in from a `data.csv` file, using your WhatsApp account.

## Running
```
git clone https://github.com/xditya/WhatsAppDMs
cd WhatsAppDMs
```
Fill up numbers in a data.csv file, in the format given in [data.csv](./data.csv).
```
npm install
npm start
```
